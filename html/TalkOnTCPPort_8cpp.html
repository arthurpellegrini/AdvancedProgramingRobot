Detailed Description

Sends a movement command to the TurtleBot via TCP.

This function constructs a movement command in a predefined format and sends it to the TurtleBot using the specified socket connection.

Parameters
socketThe socket descriptor for the TCP connection.
linear_xThe linear velocity for the TurtleBot (in meters per second).
angular_zThe angular velocity for the TurtleBot (in radians per second).
Exceptions
std::runtime_errorIf the send operation fails.

Main function to control the TurtleBot via TCP commands.

This program connects to the TurtleBot's command server and allows the user to input movement commands interactively. It also maintains the connection and handles reconnections if needed.

Parameters
argcThe number of command-line arguments.
argvThe command-line arguments. Expects the server IP and port as arguments.
Returns
Returns 0 on successful execution, or 1 on invalid input or connection failure.

Usage example:

./TalkOnTCPPort <IP> <PORT>

The program supports user input in the format <linear_x> <angular_z> to control the TurtleBot's movement. Enter q to quit the program.
