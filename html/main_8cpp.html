Detailed Description

Main program to connect to a server and handle LaserScan, Odometry, and Commander data.

This program connects to a server, receives data from multiple ports, and stores the data in shared memory for processing and debugging.

Main function of the program.

The program expects the server IP address as a command-line argument and starts threads to handle LaserScan and Odometry data while providing debugging output. It also handles Commander commands via a separate connection.
