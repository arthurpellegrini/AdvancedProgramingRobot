Listens on a TCP socket and processes incoming data. 

This function continuously receives data from the specified socket and prints it to the console with a timestamp. It also removes trailing newline characters for clean output.


\begin{DoxyParams}{Parameters}
{\em sock} & The socket descriptor for the TCP connection.\\
\hline
\end{DoxyParams}
The function reads data from the socket in a loop, processing each received message. The data is timestamped with the current local time and displayed on the console. If an error occurs during {\ttfamily recv()}, the function terminates the program.

\begin{DoxyNote}{Note}
Ensure that the socket is properly connected before calling this function. The function is designed to run in a separate thread for non-\/blocking operation. 
\end{DoxyNote}
