Listens on a specified TCP port and processes incoming connections. 

This function initializes a server socket, binds it to the specified port, and listens for incoming TCP connections. When a client connects, the function receives data from the client and displays it on the console.


\begin{DoxyParams}{Parameters}
{\em port} & The port number to listen on. \\

\end{DoxyParams}
\begin{DoxyReturn}{Returns}
Returns 0 on successful execution or a non-\/zero value if an error occurs.
\end{DoxyReturn}
\begin{DoxyNote}{Note}
Ensure that the specified port is available and not blocked by a firewall. 

This function is blocking and will run indefinitely unless an error occurs or terminated manually.
\end{DoxyNote}
The function is designed for debugging and demonstration purposes, providing a simple way to test TCP client-\/server communication. It can be extended for more complex use cases like processing received data or handling multiple clients.
