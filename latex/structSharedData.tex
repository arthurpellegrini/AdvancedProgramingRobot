\doxysubsection{Detailed Description}
Structure to store shared data for Laser\+Scan and Odometry. 

This structure holds buffers for storing Laser\+Scan and Odometry data received from the server. It is designed for use in a multi-\/threaded environment with semaphore synchronization.
